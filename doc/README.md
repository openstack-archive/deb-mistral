# Sphinx DOC hints

## Migrating from OpenStack Wiki

* Install pandoc
* Copy wiki code into a file, e.g. `source.mw`
* Convert to .rst

  pandoc --from=mediawiki --to=rst --output=doc/source/dsl/dsl_v1.rst doc/source/dsl/source.mw
